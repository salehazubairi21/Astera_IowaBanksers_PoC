1. Open Northwest_20201113 dataflow. It contains the EDI source and parser objects.
2. Open the properties of both and update the path of the source and partner profile (tpp) files that exist in this folder under Files.
3. Once the correct path has been configured you should be able preview the data flow.
4. The parser is responsible to parse the EDI message and load it into relevant transformations and destinations.


What is a tpp file?
The tpp file is the partner profile that defines the dialect and version of the incoming transaction.
